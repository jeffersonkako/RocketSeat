# Tipos de dados

Conforme o ECMAScript standart temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value
    * Structural
    * Structutal Primitive

## Primitivos

* String 
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Obejct
    * Array
    * Map
    * Set
    * Date
    * ...
* Function
